Meetify

- Meetify API docs -

1. Create User -> POST- method

Format -
{
"name" : "John Doe",
"email" : "john@gmail.com",
"dnd_start_time" : "2024-06-20T03:49:43+0000",
"dnd_end_time" : "2024-06-20T03:49:43+0000",
"preferred_timezone" : "IST"
}

2. Update User -> PUT- method

Format -
{
"dnd_start_time" : "2024-06-20T03:49:43+0000",
"dnd_end_time" : "2024-06-20T03:49:43+0000",
"preferred_timezone" : "IST"
}

3. Create Meeting -> POST- method

Format -
{
"user_id" : 130,
"meeting_type" : "online" | "offline",
"start_time" : "2024-06-20T03:49:43+0000",
"end_time" : "2024-06-20T03:49:43+0000",
"timezone" : "IST",
"notification_interval" : "0 10 * * *"
}
